# NSRI Director Profile Website   

A professional profile website for Aruzhan Maratova (me), official director at the [National Student Researchers Institute (NSRI)](https://www.linkedin.com/company/nsri-official/).   

## Overview   
This repository contains the source code for a static portfolio website. It serves to:   
* Introduce the director and their academic background.   
* Explain the mission and impact of NSRI.   
* Act as a central hub to recruit students into the NSRI global research community.   

## Technologies used   
* **HTML5:** semantic structuring.   
* **CSS3:** custom styling using CSS variables, flexbox, and CSS grid.   
* **JavaScript:** intersection observer API for smooth scrolling animations.   

## Setup & Local development   
1. Clone this repository to your local machine:   
   `git clone https://github.com/yourusername/nsri-ambassador-website.git`   
2. Open the project folder.   
3. Simply double-click `index.html` to open it in your default web browser. No local server is strictly required, though using a tool like VS Code's "Live Server" is recommended for development.   

## Deployment   
This website is ready to be hosted for free using **GitHub Pages**.     
1. Push your code to a public GitHub repository.   
2. Go to your repository settings.   
3. Navigate to **"pages"** on the left sidebar.   
4. Under **"build and deployment"**, select the `main` branch and click **"save"**.   
5. Your website will be live in a few minutes!
