# Maratova Aruzhan — NSRI School Ambassador Website

Personal ambassador page for **Maratova Aruzhan**, NSRI School Ambassador at Aqbobek International School, Aktobe, Kazakhstan.

---

## Project Structure

```
├── index.html          # Main page
├── style.css           # All styles (CSS variables, layout, animations)
├── main.js             # Scroll animations & stat counters
└── assets/
    ├── aruzhan.jpeg    # Profile photo (320×320px, optimised)
    └── nsri-logo.png   # NSRI logo, transparent background (280×280px, optimised)
```

---

## Tech Stack

- Vanilla HTML / CSS / JavaScript — no frameworks, no dependencies
- Fonts: [Playfair Display](https://fonts.google.com/specimen/Playfair+Display) + [DM Sans](https://fonts.google.com/specimen/DM+Sans) via Google Fonts
- All icons are inline SVG (no icon library required)

---

## Features

- Fully responsive (mobile-first)
- Scroll-triggered fade-up animations via `IntersectionObserver`
- Animated stat counters (ease-out cubic)
- Respects `prefers-reduced-motion`
- Optimised images — total page weight ~119KB
